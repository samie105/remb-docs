## Introduction[](#introduction "Link for Introduction ")

Learn [what React Compiler does](react/learn/react-compiler/introduction/index.md) and how it automatically optimizes your React application by handling memoization for you, eliminating the need for manual `useMemo`, `useCallback`, and `React.memo`.

## Installation[](#installation "Link for Installation ")

Get started with [installing React Compiler](react/learn/react-compiler/installation/index.md) and learn how to configure it with your build tools.

## Incremental Adoption[](#incremental-adoption "Link for Incremental Adoption ")

Learn [strategies for gradually adopting React Compiler](react/learn/react-compiler/incremental-adoption/index.md) in your existing codebase if you’re not ready to enable it everywhere yet.
