For the full table of contents (TOC), see [below](#reference-toc). The following list is a subset of the full TOC that detail core Fastify APIs and concepts in order of most likely importance to the reader:

*   [Plugins](/docs/latest/Reference/Plugins/): Explains Fastify's plugin architecture and API.
*   [Encapsulation](/docs/latest/Reference/Encapsulation/): Explains a core concept upon which all Fastify plugins are built.
