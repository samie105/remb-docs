PostgreSQL 9.6 and below provided a command named `pg_receivexlog` to fetch write-ahead-log (WAL) files. This command was renamed to `pg_receivewal`, see [pg\_receivewal](https://www.postgresql.org/docs/current/app-pgreceivewal.html "pg_receivewal") for documentation of `pg_receivewal` and see [the release notes for PostgreSQL 10](https://www.postgresql.org/docs/current/release-prior.html "E.5. Prior Releases") for details on this change.
