* * *

PostgreSQL allows user-defined functions to be written in other languages besides SQL and C. These other languages are generically called _procedural languages_ (PLs). Procedural languages aren't built into the PostgreSQL server; they are offered by loadable modules. See [Chapter 40](https://www.postgresql.org/docs/current/xplang.html "Chapter 40. Procedural Languages") and following chapters for more information.
