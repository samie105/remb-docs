This is documentation for

tRPC

**9.x**, which is no longer actively maintained.

For up-to-date documentation, see the **[latest version](https://trpc.io/docs/)** (

11.x

).

Version: 9.x

All contributions are welcome and appreciated, big or small!

Feel free to submit pull requests, create issues, or start discussions on the [GitHub repo](https://github.com/trpc/trpc). If you are not sure where to start, we have a tag for [good first issues](https://github.com/trpc/trpc/issues?q=is%3Aopen+is%3Aissue+label%3A%22%F0%9F%91%89+good+first+issue%22).

If you need help or feedback, contact [**@alexdotjs** on Twitter](https://twitter.com/alexdotjs) or write in our [Discord Server](https://trpc.io/discord)!

See official contribution guide [here on GitHub](https://github.com/trpc/trpc/blob/main/CONTRIBUTING.md).
