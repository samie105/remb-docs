## Overview

Provide helpers with asynchronous tasks, like [`delay`](https://jsr.io/@std/async@1.2.0/doc/~/delay), [`debounce`](https://jsr.io/@std/async@1.2.0/doc/~/debounce), [`retry`](https://jsr.io/@std/async@1.2.0/doc/~/retry), or [`pooledMap`](https://jsr.io/@std/async@1.2.0/doc/~/pooledMap).

```js
import { delay } from "@std/async/delay";

await delay(100); // waits for 100 milliseconds
```

### Add to your project

\>\_

```sh
deno add jsr:@std/async
```
